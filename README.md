# MotoApp POI Data

Point of Interest (POI) data for motorcycle navigation apps.

## Data Sources

All data is sourced from [OpenStreetMap](https://www.openstreetmap.org/), processed and organized by country for efficient delivery.

## Available Datasets

### Fuel Stations

- **USA**: 113,442 fuel stations
- **Canada**: 5,765 fuel stations  
- **Mexico**: 8,345 fuel stations

### Coming Soon

- European countries (Spain, France, Italy, Germany, etc.)
- Additional POI categories (restaurants, hotels, camping, etc.)
- Twisty roads and mountain passes

## Usage

These GeoJSON files can be fetched directly via GitHub raw URLs:

```
https://raw.githubusercontent.com/LordGomez/MotoApp-POI-Data/main/us-fuel-stations.json
https://raw.githubusercontent.com/LordGomez/MotoApp-POI-Data/main/ca-fuel-stations.json
https://raw.githubusercontent.com/LordGomez/MotoApp-POI-Data/main/mx-fuel-stations.json
```
